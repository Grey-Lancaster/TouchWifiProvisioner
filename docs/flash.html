Try it in your browser

Flash a ready-built demo straight from Chrome or Edge over USB — no Arduino IDE, no PlatformIO, nothing to install.

Before you start: use desktop Chrome or Edge (Web Serial isn't supported on Firefox/Safari or mobile), and connect your board over USB. This flashes a small demo sketch that shows the picker UI and, once connected, a green screen with the network name and IP — it will overwrite whatever is currently on the board.

Cheap Yellow Display

ESP32-2432S028R · ST7789 panel

The classic 2.8" resistive-touch CYD. This flashes the CYD_BasicConnect example.

Your browser doesn't support Web Serial - use desktop Chrome or Edge. This page needs to be loaded over HTTPS to flash (it already is here).
Assumes the ST7789 panel variant - see the example README if your unit turns out to be ILI9341-based.
